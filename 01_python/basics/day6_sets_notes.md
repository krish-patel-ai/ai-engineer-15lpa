# Sets in Python

## What is a set?
A set is a collection of unique values. Duplicate values are removed automatically.

## Why sets exist
Sets provide fast lookup and ensure uniqueness.

## When to use sets
When order does not matter and duplicates should not exist.

## When NOT to use sets
When order matters or duplicates are meaningful.

## Common interview questions
Why not use sets everywhere?
Because sets remove order and duplicates, which can cause bugs.
